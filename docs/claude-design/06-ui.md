# UI

## Ranking page

Source tab strip + period strip + grouped sections. Period options auto-adjust
to the selected scope's supported periods (e.g. selecting AlphaPolis hides
daily/weekly/monthly and shows just `hot`).

### Single-source view (Syosetu)

![Syosetu ranking](screenshots/ranking-syosetu.png)

Source pill `なろう` on every row (and on the №1 hero). The legacy syosetu+
(daily|weekly|monthly|quarterly) path returns `{ items: [] }` and the page
normalizes it into a one-section view, so this view doesn't render the
section header above the list.

### Single-source view (Kakuyomu)

![Kakuyomu ranking](screenshots/ranking-kakuyomu.png)

Period strip now shows daily / weekly / monthly / yearly / 전체기간 (entire).
`カクヨム` source pill on every row. Total-episodes column shows `—` because
the ranking page doesn't carry that field on kakuyomu — the value lands on
register, not on listing.

### Single-source view (AlphaPolis)

![AlphaPolis ranking](screenshots/ranking-alphapolis.png)

Period strip collapses to just `인기` (hot). `α` source pill in amber.
Numeric composite IDs are visible in the hero card (`262615050/530048750`).

### Combined "All" view

![All-sources grouped ranking](screenshots/ranking-all-grouped.png)

Three sections stacked: Syosetu, Kakuyomu, AlphaPolis. Each section has its
own header pill + item count. Adult sites (Nocturne) only appear when the
user has the adult toggle on.

## Source pill component

`src/components/source-pill.tsx`. Tiny presentational component used in the
ranking row, ranking hero, and (eventually) novel hero. Color-coded by site
to give the eye a quick parse:

| Site | Short label | Color |
|---|---|---|
| Syosetu | なろう | emerald |
| Nocturne | ノクタ | rose |
| Kakuyomu | カクヨム | sky |
| AlphaPolis | α | amber |

```tsx
<SourcePill site="kakuyomu" />          // 短 (なろう / ノクタ / カクヨム / α)
<SourcePill site="kakuyomu" variant="full" />  // Full (Syosetu / Nocturne / Kakuyomu / AlphaPolis)
```

## Settings page — adult toggle

![Adult toggle in Reading section](screenshots/settings-adult-toggle.png)

Adult toggle sits in the Reading section between the reader-typography hint
and the theme picker. Hint text explains it gates Nocturne plus
AlphaPolis adult-flagged works in ranking, search, and library. PUT to
`/api/settings` is fire-and-forget on change.

## Register page

![Register page](screenshots/register-page.png)

The page-level copy still says "syosetu URL or ncode" (left for follow-up i18n
update). The actual `parseInput` validator accepts URLs and bare IDs from any
of the four sources — paste a kakuyomu URL or alphapolis composite ID and
register works end-to-end. The placeholder updates would land in a follow-up
i18n pass alongside any other Korean copy that still mentions "なろう" by
name.
